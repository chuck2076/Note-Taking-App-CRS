# Note Taker Using Express.js

## User Story

```
AS A small business owner
I WANT to be able to write and save notes
SO THAT I can organize my thoughts and keep track of tasks I need to complete
```

## Description:
```
 This app uses Node.js and Express.js to generate a note taking web page so the User can write, save and delete notes.  

 Existing notes are listed in the left-hand column with empty fields on the right-hand side for the new noteâ€™s title and text.

 ```
